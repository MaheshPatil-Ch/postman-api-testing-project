# REST API Testing using Postman

## Project Overview
This project demonstrates REST API testing using Postman and ReqRes APIs.

## Features
- GET API testing
- POST API testing
- PUT API testing
- DELETE API testing
- JSON response validation
- Status code verification

## Tools Used
- Postman
- ReqRes API
- JavaScript

## API Endpoints Tested
- GET /users
- POST /users
- PUT /users/{id}
- DELETE /users/{id}

## Purpose
The purpose of this project is to learn REST API testing and automation basics using Postman.
